Sistema simples de cadastro usando Python e banco de dados SQLite
# Cadastro de Clientes – Python + SQLite

Projeto prático desenvolvido para consolidar conhecimentos em Python e SQL,
utilizando banco de dados SQLite.

## Funcionalidades implementadas
- Cadastro (CREATE)
- Listagem (READ)
- Atualização (UPDATE)
- Exclusão (DELETE)
- Validação básica de e-mail

## Tecnologias
- Python
- SQLite
- SQL
- Git/GitHub

## Como executar
1. Clone o repositório
2. Execute o arquivo `app.py`
3. Utilize o menu no terminal

CRUD completo – versão funcional
